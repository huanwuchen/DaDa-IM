## 三、调用链路

### 3.1 消息发送链路

```
用户输入文字
    │
    ▼
WxChatViewModel.sendTextMessage()
    │
    ▼
SendMessageUseCase.sendText(myUserId, peer, content)
    │
    ▼
ImChatRepositoryImpl.sendTextMessage()
    ├── 1. 创建 ImMessageEntity → Room INSERT (本地持久化)
    ├── 2. 创建 ImConversationEntity → Room UPSERT (更新会话)
    └── 3. 构建 MessageModel → WebSocketManager.sendMessage()
                                      │
                                      ▼
                               OkHttp WebSocket.send()
                                      │
                                      ▼
                                IM Server 转发
                                      │
                                      ▼
                               对方客户端接收
```

### 3.2 消息接收链路

```
OkHttp WebSocket 收到文本帧
    │
    ▼
WebSocketListenerImpl.onMessage()
    ├── onRawText(text) → MessageManager.handleInbound()
    │                          ├── 解析 Frame 信封
    │                          ├── 去重 (LruDedup)
    │                          ├── 落库 (MessageStore)
    │                          ├── 回 ACK
    │                          └── _inbound.tryEmit(message)
    │
    └── onMessage(message) → 旧监听器分发
                               ├── GlobalMessagePersister.onMessageReceived()
                               │       └── Room INSERT (消息+会话)
                               ├── GlobalMessageNotifier.onNewMessage()
                               │       ├── 悬浮窗通知
                               │       └── 系统通知栏
                               └── WxChatViewModel (如果在聊天页)
                                       └── UI 更新
```

### 3.3 音视频通话链路

```
用户点击"语音通话"
    │
    ▼
CallManager.invite(targetUserId, CallType.AUDIO)
    ├── 1. 预分配 UDP Socket (audioPort)
    ├── 2. 构建 CallPayload { callId, callType, transport:"udp", data:{audioIp, audioPort} }
    ├── 3. WebSocket 发送 { type:"call-invite", content: JSON(CallPayload) }
    └── 4. callState → CALLING

对方收到 call-invite
    │
    ▼
WebSocketService → CallManager.handleSignal()
    ├── 1. 解析 MediaEndpoints
    ├── 2. 预分配本地 UDP Socket
    ├── 3. WebSocket 回复 { type:"call-accept", content: JSON(本地端点) }
    └── 4. VoiceEngine.start(remoteIp, remotePort)

发起方收到 call-accept
    │
    ▼
VoiceEngine.start(remoteIp, remotePort)
    ├── sendThread: AudioRecord → UDP Send
    └── recvThread: UDP Recv → AudioTrack
```

## 四、数据流向

### 4.1 数据存储架构

```
┌─────────────────────────────────────────────┐
│                 数据存储层                     │
│                                              │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  │
│  │  Room    │  │  MMKV    │  │  Server  │  │
│  │ (SQLite) │  │ (KV)     │  │ (Remote) │  │
│  └────┬─────┘  └────┬─────┘  └────┬─────┘  │
│       │              │              │        │
│  im_messages     access_token    /api/*     │
│  im_conversations user_id       /ws/*       │
│  im_contacts     device_id                   │
│  im_user_profile cover_image                 │
│  chat_messages                               │
└─────────────────────────────────────────────┘
```

### 4.2 数据流向图

```
发送消息:
  UI → ViewModel → UseCase → Repository
    ├── Room INSERT (im_messages + im_conversations)
    └── WebSocket SEND → Server → 对方

接收消息:
  WebSocket RECV → MessageManager (去重+ACK)
    ├── GlobalMessagePersister → Room INSERT
    ├── GlobalMessageNotifier → 通知
    └── ViewModel → UI 更新

登录流程:
  UI → ImApiService.register() → Server
    ├── 返回: userId, token, username, avatar
    ├── MMKV 保存: token, userId, deviceId
    ├── Room 保存: ImUserProfileEntity
    └── WebSocket 连接 + JPush 别名绑定

文件上传:
  UI → SendMessageUseCase → MediaUploader
    ├── OkHttp Multipart POST /api/upload
    ├── 返回: { url, type, fileName, size }
    └── Repository: 构建 MessageModel(content=url) → WebSocket 发送
```

## 五、Hilt 依赖注入结构

```
NetworkModule (@InstallIn SingletonComponent)
  ├── OkHttpClient (带 AuthInterceptor)
  ├── Retrofit (baseUrl = Constants.BASE_URL)
  ├── ImApiService
  ├── UserApiService
  ├── MomentApiService
  ├── MessageStore (Noop 默认实现)
  └── MessageManager

DataModule (@InstallIn SingletonComponent)
  ├── AppDatabase (Room)
  ├── ChatMessageDao
  ├── ImConversationDao
  ├── ImMessageDao
  ├── ImContactDao
  ├── ImUserProfileDao
  ├── UserPreferences (→ UserPreferencesImpl)
  └── ChatLocalDataSource

RepositoryModule (@InstallIn SingletonComponent)
  ├── ImChatRepository → ImChatRepositoryImpl
  ├── ImUserRepository → ImUserRepositoryImpl
  ├── ImContactRepository → ImContactRepositoryImpl
  ├── MomentRepository → MomentRepositoryImpl
  ├── UserProfileRepository → UserProfileRepositoryImpl
  ├── FriendRepository → FriendRepositoryImpl
  └── AiChatRepository → AiChatRepositoryImpl

ImageLoaderModule (@InstallIn SingletonComponent)
  └── ImageLoader → GlideImageLoader
```
