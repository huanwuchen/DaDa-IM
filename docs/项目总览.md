# DaDa IM 项目总览

## 一、项目定位

**项目名称**: DaDa (DaDaNew)

**项目类型**: 即时通讯 (IM) + 社交应用，对标微信，属于 **IM + 社交 + 音视频** 复合类型。

**解决的问题**: 提供一个完整的移动端即时通讯解决方案，包含：
- 文字/图片/语音/视频/文件 消息收发
- 朋友圈社交功能
- 音视频通话（自研 UDP 引擎 + 腾讯云 TUICallKit 双方案）
- AI 智能助手（流式对话）
- 好友管理、通讯录、推送通知

**技术定位**: Android 原生 IM 应用，采用多模块 Clean Architecture + MVVM 架构。

## 二、技术栈全景

### 客户端

| 类别 | 技术 | 版本 |
|------|------|------|
| 语言 | Kotlin | 1.8.22 |
| 构建 | Gradle + AGP | 7.5.1 / 7.4.2 |
| DI | Hilt (Dagger) | 2.48 |
| 网络 | Retrofit + OkHttp | 2.9.0 / 4.11.0 |
| 实时通信 | OkHttp WebSocket | 4.11.0 |
| 数据库 | Room | 2.5.2 |
| KV 存储 | MMKV (腾讯) | 1.3.3 |
| 图片加载 | Glide | 4.15.1 |
| 异步 | Kotlin Coroutines + Flow | 1.7.3 |
| UI | XML + ViewBinding + Lottie | - |
| 媒体播放 | ExoPlayer (Media3) | 1.3.1 |
| 二维码 | ZXing | 4.3.0 |
| 推送 | 极光 JPush | 5.5.0 |
| 音视频 | 腾讯 TUICallKit + TRTC | 2.7.0 / 12.5.0 |
| 自研音视频 | UDP 直连引擎 | 自研 |
| AI 对话 | MiMo API (SSE 流式) | - |

### 服务端（推断自 API 接口）

| 类别 | 技术 |
|------|------|
| 协议 | HTTP RESTful + WebSocket |
| 地址 | `http://192.168.124.17:8080` (局域网) |
| WebSocket | `ws://192.168.124.17:8080/ws/{userId}` |
| 文件上传 | Multipart, POST `/api/upload` |
| 推送网关 | 极光推送服务端 SDK |

### 第三方服务

| 服务 | 用途 |
|------|------|
| 极光推送 (JPush) | 离线推送、别名绑定 |
| 腾讯云 TRTC | 音视频通话（TUICallKit 方案） |
| MiMo API | AI 智能助手（流式 SSE） |
| 阿里云 Maven | 依赖仓库镜像 |

## 三、模块结构

```
DaDaNew/
├── app/                    # 主模块：UI、业务逻辑、Repository 实现
│   ├── data/repository/    # Repository 实现（7 个）
│   ├── di/                 # Hilt 模块（RepositoryModule）
│   ├── domain/             # UseCase（SendMessageUseCase）
│   ├── network/
│   │   ├── call/           # 音视频通话（CallManager、VoiceEngine、VideoEngine）
│   │   └── websocket/      # WebSocket 服务（WebSocketService、GlobalMessagePersister）
│   ├── notification/       # 通知管理（3 个）
│   ├── push/               # 极光推送（PushManager、JPushReceiver）
│   ├── ui/                 # 页面（16 个功能模块）
│   ├── utils/              # 工具类
│   └── widget/             # 自定义控件（Emoji、ImageViewer、VideoPlayer）
├── domain/                 # 领域层：Repository 接口、领域模型
│   ├── aichat/repository/  # AiChatRepository 接口
│   ├── chat/               # 聊天模型和接口
│   ├── contact/repository/ # 通讯录接口
│   ├── friend/repository/  # 好友接口
│   ├── moment/repository/  # 朋友圈接口
│   └── user/repository/    # 用户接口
├── core/
│   ├── common/             # 基础工具（Constants、LogUtil、KvUtil、BaseViewModel）
│   ├── database/           # Room 数据库（5 Entity、5 DAO、UserPreferences）
│   ├── network/            # 网络层（Retrofit API、WebSocket Manager、MessageManager）
│   ├── imageloader/        # 图片加载封装（Glide）
│   └── ui/                 # UI 基类（BaseActivity、BaseFragment）
└── docs/                   # 项目文档
```

## 五、功能清单

| 功能模块 | 核心能力 |
|---------|---------|
| IM 单聊 | 文字/图片/语音/视频/文件消息，已读回执，未读计数 |
